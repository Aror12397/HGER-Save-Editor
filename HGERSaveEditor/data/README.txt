== data 폴더 사용 방법 ==

이 폴더의 텍스트 파일들이 없으면 프로그램은 "종족 #N" 형태의 기본 이름을 사용합니다.

■ species.txt
  - 포켓몬 종족 이름 목록 (한 줄에 하나씩)
  - 0번째 줄 = "없음" (빈 슬롯)
  - 1번째 줄 = 이상해씨, 2번째 줄 = 이상해풀, ...
  - hg-engine 추가 포켓몬은 493번 이후 줄에 추가

■ moves.txt
  - 기술 이름 목록 (한 줄에 하나씩)
  - 0번째 줄 = "없음"
  - 1번째 줄 = 박치기, ...
  - hg-engine 추가 기술은 기존 목록 이후에 추가

■ items.txt
  - 아이템 이름 목록 (한 줄에 하나씩)

■ abilities.txt
  - 특성 이름 목록 (한 줄에 하나씩)

■ basestats.csv
  - 기본 스탯 데이터 (스탯 계산에 사용)
  - 형식: species_id,HP,Atk,Def,SpA,SpD,Spe
  - 예: 1,45,49,49,65,65,45
  - 이 파일이 없으면 스탯 계산을 지원하지 않음

■ charmap.txt
  - HGER 게임 전용 문자 인코딩 테이블
  - 없으면 닉네임/트레이너 이름/박스 이름이 표시되지 않음
  - 형식: HEXCODE=문자 (예: 0400=가)
