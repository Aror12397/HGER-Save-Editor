# HGER Save Editor

hg-engine 기반 포켓몬 롬 핵(HGER)의 `.sav` 파일을 편집하는 Windows 애플리케이션.

## 다운로드

[최신 릴리스](https://github.com/Aror12397/HGER-Save-Editor/releases/latest)에서 zip 파일을 받아 압축 해제 후 실행하세요.

실행 전 [.NET 10 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/10.0) 설치가 필요합니다.

## 기능

- **파티 / 박스 편집** — 6슬롯 파티, 18박스 × 30슬롯 박스 지원
- **포켓몬 편집** — 종류, 폼, 닉네임, 레벨, 경험치, 특성, 지닌 아이템, 친밀도, 볼, 언어
- **기술 편집** — 4슬롯 기술 ID, PP, PP업
- **스탯 편집** — IV, EV, 계산 스탯 표시 (IV 최대, EV 초기화 버튼)
- **만남 정보** — 어버이 이름, TID/SID, 만난 레벨, 만난 장소, 만난 날짜
- **트레이너 정보 표시** — 이름, 성별, TID (읽기 전용)
- **스프라이트 표시** — 폼별 스프라이트 지원
- **드래그 앤 드롭** — `.sav` / `.dsv` 파일을 끌어다 놓으면 바로 열기
- **PID 기반 자동 표시** — 성격, 성별, 이로치 여부를 PID에서 자동 판별

## 데이터 파일 커스터마이즈

`data/` 폴더의 텍스트 파일을 직접 수정하여 hg-engine 추가 포켓몬/기술에 대응할 수 있습니다.
각 파일은 줄 번호 = ID 방식이며, 493번 이후 줄에 추가 포켓몬을 이어 붙이면 됩니다.
